# bash
## if
### [] 和 [[ ]] 的区别

尽量不用 []，建议用 [[ ]]。

``` bash
if [ ... ]; then
  ...
fi
```

``` bash
if [[ ... ]]; then
  ...
fi
```

Shell 中的 test 命令用于检查某个条件是否成立，它可以进行数值，字符和文件三个方面的测试。

if test 等价于 if [ ]， [[ ]] 于后期版本才支持，功能更强大。

#### 相同处
- 两者都建议在表达式和括号自身使用空格避免出错。

  注意空格
  ``` bash
  if [[ $1 != "start" && $1 != "stop" ]]
  ```
