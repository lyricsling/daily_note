# Design patterns
设计模式带来的好处。
设计准则。
设计模式如何运用于实际。

code reuse, experience reuse。

1. 
  找出程序中的变量和不变量。

  让变化的部分和不变化的部分独立出来。

  改变代码后更不容易出现错误，以及让代码扩展更灵活。


2. 
  Program to an interface, not an implementation。
